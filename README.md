# 尚医通后台管理项目

## 启动项目

**建议使用 yarn 下载依赖**

1. 安装 yarn

`npm i yarn -g`

2. 配置 yarn 的淘宝镜像

`yarn config set registry https://registry.npmmirror.com/`

原来淘宝镜像地址即将失效，注意更换新的。

> [淘宝 npm 域名即将切换](https://zhuanlan.zhihu.com/p/465424728)

3. 下载依赖

**注意来到项目根目录**

`yarn`

4. 运行项目

运行项目的命令都在 package.json 中 scripts 里面

- `yarn start` 启动开发环境
- `yarn build` 启动生产环境

> 更多关于项目的介绍，请看项目文档~
